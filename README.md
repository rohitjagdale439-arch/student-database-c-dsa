# 📚 Student Database C DSA

### 🔹 Data Structures & File Handling Project in C

## 🔹 Overview

This project is a menu-driven Student Database system developed in C using singly linked lists. It allows users to efficiently manage student records with dynamic memory allocation and file handling.

## 🔹 Features

* ➕ Insert Student Record
* ❌ Delete Student Record
* 🔍 Search Student
* ✏️ Update Student Details
* 📄 Display All Records
* 💾 Save & Load Data

## 🔹 Technologies Used

* C Programming
* Data Structures (Singly Linked List)
* File Handling

## 🔹 Project Structure

```id="str1"
student-database-c-dsa/
```
